zipcode Cloud Relay

A Gemini field coding agent for low-connectivity teams. The local agent gathers bounded source evidence, produces a human-reviewable patch plan, and then relays approved work into Google Cloud Agent Builder and GitLab MCP when connectivity returns.

1. Local Evidence

Inspect a selected workspace with file listing, file reading, and search while blocking reads outside the project root.

2. Gemini Plan

Use approved evidence to produce a grounded fix plan, tests, and an audit transcript under human oversight.

3. GitLab Relay

Prepare issues, merge request context, review summaries, and status updates through the GitLab MCP track.

python zipcode_agent.py \
  --root examples/airgap_demo_workspace \
  --task "Find the bug in the water sensor parser and propose a safe fix." \
  --read README.md \
  --search TODO \
  --dry-run
